# Hugging Face Release Plan

- Public run ledger JSON and bundle manifests generated from toy/downscaled
  trace JSONL. The current checked-in bundle set is
  `code_based_toy_classic_mceliece_v0`, `code_based_toy_hqc_v0`,
  `code_based_toy_isd_v0`, `code_based_toy_mdpc_v0`,

  `implementation_security_toy_kat_v0`,

  `lattice_downscaled_lwe_instance_solve_v0`,
  `lattice_downscaled_mlwe_instance_solve_v0`,

- No real private evolution traces.

## Space

Suggested private-first Agent Environment Space:
`agades/agades-pqc-gym-agent-env`

The Space should allow users to select a toy/schema-only AttackPlan, run validation, and view mock or unsupported evaluator status. It must not execute arbitrary code.

The repository includes `hf/README.md`, `hf/app.py`, `hf/requirements.txt`, and
the checked-in `hf/dataset/` bundle as the executable Space skeleton. The app
reads valid public AttackPlans from `hf/dataset/attack_plans.jsonl`, calls the
shared public verifier API, and keeps arbitrary Python execution disabled. The
requirements file installs `agades-pqc-gym` from the public GitHub repo so the
Space does not depend on a local `src/` checkout, and it must remain compatible
with the Gradio version injected by Hugging Face Spaces.

`hf/space_manifest.json`, generated by `agades-pqc hf-space-manifest`, is the
Space publication contract: it records the exposed public example labels,
default example, dependency file, shared verifier schema, and no-code/no-claim
boundaries.

`docs/huggingface_publication_handoff.json`, generated by
`agades-pqc hf-publication-handoff`, is the consolidated local handoff for
dataset, Space, and Collection review. It records local artifact digests,
review checklist items, credential boundaries, and the explicit invariant that
no Hugging Face Hub publication has been performed by the generator.

The Space summary and dataset cards must state that verifier output is not a
security claim.

Use a private Space first during release review:

```bash
hf auth login
hf repo create agades/agades-pqc-gym-agent-env --repo-type=space --space_sdk gradio --private --exist-ok
hf upload agades/agades-pqc-gym-agent-env hf . --repo-type=space --commit-message "Sync Agades PQC Gym Agent Environment"
```

For CI or release automation, prefer `HF_TOKEN` in the environment rather than
passing tokens on the command line. Public visibility requires release review
of `hf/space_manifest.json`, `hf/README.md`, `hf/dataset/`, `hf/requirements.txt`, and the
no-claim Space summary.

## Local Export

The checked-in `hf/dataset/` directory is generated by:

```bash
uv run agades-pqc hf-dataset --out hf/dataset
uv run agades-pqc hf-space-manifest --out hf/space_manifest.json
uv run agades-pqc hf-collection-manifest --out hf/collection_manifest.json
uv run agades-pqc hf-publication-handoff --out docs/huggingface_publication_handoff.json
uv run agades-pqc hf-publication-handoff-verify --handoff docs/huggingface_publication_handoff.json
```

It contains deterministic JSONL rows for public AttackPlans, shared
`agades.pqc.task_metadata.v4` task constraints, seed AttackPlan SHA-256
digests, seed verifier status/reward, seed estimator, seed reproduction status,
matching verifier outputs, all current public run bundles, and a root checksum
manifest.

`docs/source_catalog.json`, `docs/benchmark_source_contracts.json`,
`docs/public_benchmark_manifest.json`, and `public/run_export/manifest.json`
should be published or linked beside the
dataset/Space when describing external anchors and the local public benchmark
v0 bundle set. The source catalog is a source map, and the benchmark source
contracts are review gates for heavyweight adapters such as TAPAS and
LWE-benchmarking, future instruction/evaluation anchors such as
`AYI-NEDJIMI/post-quantum-crypto-fr`,
`AYI-NEDJIMI/post-quantum-crypto-en`, `Q-GRID/pqc-ssl-scans`, and
`SCA-HNUST/SC2026`, and future implementation anchors such as PQ Code Package,
liboqs, PQClean, dudect, ctgrind, or TIMECOP/SUPERCOP. None of these contracts
is a claim that the source is implemented as a current verifier, training, or
benchmark surface.
It also records Prime ecosystem anchors for public release reviewers:
`https://app.primeintellect.ai/dashboard/home/quickstart` for onboarding,
`https://www.primeintellect.ai/auto-nanogpt` for the auto-nanoGPT public
benchmark story, and `PrimeIntellect-ai/experiments-autonomous-speedrunning`
for the released scratchpads/run logs/config pattern. These are source-map-only
references for public evaluator observability, not a claim that Agades publishes
private evolution traces or runs a Prime training job from the Hugging Face
surface.

Before publishing, regenerate `public/release_audit.json` and verify the
`hf-dataset-safety`, `hf-space-manifest`, `hf-collection-manifest`, and
`hf-publication-handoff` checks remain `passed`. Then run the private-run
policy and local publication preflight:

```bash
uv run agades-pqc private-run-policy --out docs/private_run_policy.json
uv run agades-pqc private-run-policy-verify --policy docs/private_run_policy.json
uv run agades-pqc publication-preflight --out public/publication_preflight.json
uv run agades-pqc publication-preflight-verify --preflight public/publication_preflight.json
uv run agades-pqc external-publication-review-packet --out docs/external_publication_review_packet.json
uv run agades-pqc external-publication-review-packet-verify --packet docs/external_publication_review_packet.json
```

The checked-in `docs/private_run_policy.json` and
`public/publication_preflight.json`, summarized by
`docs/external_publication_review_packet.json`, keep external publication
blocked from private traces, prompts, evaluator weights, and unreleased
candidates until release review and credential review are both approved.

## Collection

Suggested collection: `Agades PQC Gym`

Include the GitHub repo, toy dataset, Space, benchmark card, and source map.
The checked-in machine-readable collection contract is
`hf/collection_manifest.json`, generated by:

```bash
uv run agades-pqc hf-collection-manifest --out hf/collection_manifest.json
```

The collection manifest links the GitHub repository, suggested dataset
`agades/pqc-gym-toy`, suggested Space `agades/pqc-gym`, `hf/benchmark_card.md`,
`docs/source_catalog.json`, `docs/public_benchmark_manifest.json`, and
`public/run_export/manifest.json`. The Hugging Face publication handoff records
the checked local bundle digests and review checklist across the same dataset,
Space, and Collection surfaces. Both artifacts are publication checklists only:
they require release review and must not publish private evolution traces,
candidate strategies, credentials, or security claims.

## Private Holdback

Do not publish real traces, prompts, evaluator weights, unpublished candidate strategies, private reports, or collaborator correspondence.
