# Hugging Face Release Plan

## Public Dataset

`agades/lwe-attackplan-toy`

Contents:

- Toy/downscaled AttackPlan JSON files.
- Valid and invalid examples.
- Mock evaluator outputs.
- Dataset card explaining limitations and no-claim boundaries.

## Space

`agades/lwe-strategy-gym`

The Space should expose a small UI where users select a toy target and operator, then view validation and mock-estimator output with responsible research disclaimers.

## Holdback

Do not publish real traces, prompts, evaluator weights, best candidate strategies, unpublished reports, or collaborator correspondence.
