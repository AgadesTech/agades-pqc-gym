# NVIDIA And Accelerator Strategy

Agades should present this repository as a serious open-core research substrate for evaluator-driven cryptanalysis, not as a claim generator.

## NVIDIA Fit

NVIDIA Inception is relevant because Agades is an AI/security startup building evaluator-driven systems. The near-term open-source surface should emphasize:

- deterministic verifier environments;
- run ledgers and observability;
- GPU-ready future workloads for LWE/TAPAS experiments and reviewed
  side-channel reproduction planning;
- agent evaluation harnesses;
- careful separation between public benchmark artifacts and private evolution traces.

## Agent/Evaluation Fit

NVIDIA's 2026 agent and open-model direction makes the strongest public story about evaluation harnesses, safety gates, and observable long-horizon agent behavior. Agades can fit that surface by publishing:

- typed AttackPlan environments;
- verifier specs;
- reproducible toy traces;
- redaction tooling;
- benchmark reports with clear limitations.

summary for accelerator-facing review. It is generated by:

```bash
uv run agades-pqc nvidia-manifest --out nvidia/accelerator_manifest.json
uv run agades-pqc nvidia-manifest-verify --manifest nvidia/accelerator_manifest.json
```

## Holdback

Do not publish proprietary traces, prompt policies, evaluator weights, unpublished attack candidates, or private collaborator materials for accelerator visibility.
