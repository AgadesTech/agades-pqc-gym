# Source Map

This map records public anchors used by the project. It is not an endorsement and does not replace expert review.

The machine-readable form is `docs/source_catalog.json`, generated by:

```bash
uv run agades-pqc source-catalog --out docs/source_catalog.json
```

That JSON artifact is intended for release review, Hugging Face/Prime/NVIDIA packaging, and future benchmark-adapter planning. It does not include private evolution traces, unpublished candidate strategies, private prompts, or responsible-disclosure material.

Heavyweight benchmark sources that could require large storage, GPUs, external
toolchains, or expert parameter mapping are separately bounded by
`docs/benchmark_source_contracts.json`, generated by:

```bash
uv run agades-pqc benchmark-source-contracts --out docs/benchmark_source_contracts.json
uv run agades-pqc benchmark-source-verify --contracts docs/benchmark_source_contracts.json
```

Those contracts are review gates for future adapters, not active public verifier
integrations or cryptanalytic claims.
