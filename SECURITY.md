# Security Policy

Agades PQC Gym is research tooling for public, toy, downscaled, and authorized settings.

Do not use this repository to target live third-party systems. Do not submit malware, persistence, evasion, exfiltration, credential theft, or weaponized exploit chains.

If a report identifies a potential issue in a real implementation or standard setting, treat it as responsible-disclosure material and do not publish it through this repository before review.
