# Dataset Card: Agades LWE AttackPlan Toy

## Summary

This dataset contains toy and downscaled AttackPlan JSON examples for evaluator-driven LWE/MLWE strategy search.

## Contents

- Valid toy AttackPlans.
- Invalid examples for validator tests.
- Mock-estimator outputs for reproducibility demos.

## Limitations

The dataset contains no private evolution traces and makes no security claims about deployed cryptographic standards.
