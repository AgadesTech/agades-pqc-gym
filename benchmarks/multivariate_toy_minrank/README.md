# Multivariate Toy MinRank Benchmark

This benchmark exercises the bounded `MULTIVARIATE` toy MinRank evaluator on small public `GF(2)` matrix pencil shapes.

It includes rank-0 and rank-1 reproduction fixtures under `fixtures/`, solved by bounded exhaustive search only to verify public reproducibility plumbing.

It is a verifier-plumbing fixture only. It is not a MinRank attack implementation for deployed schemes, not a UOV/MAYO/Rainbow security claim, and not evidence about real multivariate parameters.
