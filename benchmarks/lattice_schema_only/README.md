# Lattice Schema-Only Boundary Benchmarks

This benchmark folder contains public schema-only NTRU and SIS targets for the
`lattice` plugin boundary.

These seeds are routing and applicability checks only. They deliberately return
`evaluation_status="unsupported"` until reviewed NTRU or SIS estimator mappings
exist, and they must not be interpreted as Lattice Estimator claims or deployed
parameter security evidence.
